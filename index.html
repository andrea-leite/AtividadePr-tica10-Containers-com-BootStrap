Construindo uma página Web Responsiva

Objetivos

Esta atividade tem como objetivo permitir que você pratique e aplique o conhecimento adquirido durante a aula sobre a implementação de Containers utilizando o framework Bootstrap.

Materiais, Métodos e Ferramentas

Um editor de texto de sua escolha (ex: Visual Studio Code, Sublime Text, Atom, etc.), navegador web e Bootstrap (Você pode linkar o Bootstrap via CDN ou fazer o download da biblioteca para o seu projeto)

Atividade Prática

  • Crie uma página Web que inclua um cabeçalho, um corpo e um rodapé.
  • No cabeçalho, adicione o nome da sua página web e um menu de navegação com pelo menos três itens.
  • No corpo da página, adicione três seções diferentes usando containers do Bootstrap. Cada seção deve incluir um título e um parágrafo de texto.
  • No rodapé, adicione um copyright com o ano atual.
  • Aplique estilos à sua página usando classes do Bootstrap, como por exemplo as classes para cores de fundo e texto.
  • Certifique-se de que sua página é responsiva e se adapta a diferentes tamanhos de tela.
